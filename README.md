# ClawVoice 🎙️

> Open source voice-to-text for any app, powered by Claude AI.

**No subscription. No backend. Just your Claude API key.**

ClawVoice is a lightweight Android app that lets you dictate text into any app on your phone — email, messages, notes, browsers, everything — using Claude AI for transcription and cleanup.

## How it works
1. Tap the floating mic button (or use volume button shortcut)
2. Speak naturally
3. ClawVoice transcribes using Claude and injects the text into whatever field you're typing in

## Why open source?
Because voice dictation should be free infrastructure, not a $16/month subscription.

Bring your own Claude API key. Your audio never touches our servers — it goes directly from your phone to Anthropic's API.

## Features
- 🎙️ Works in ANY app (email, SMS, browsers, notes, everything)
- 🤖 Claude-powered transcription + text cleanup
- 🔒 Privacy-first: no servers, no accounts, your API key only
- ⚡ Floating button overlay — one tap to start, one tap to stop
- 🌍 Multi-language support (via Claude)
- 📱 Android first, Windows coming soon

## Setup
1. Get a Claude API key at console.anthropic.com
2. Install ClawVoice
3. Enter your API key in settings
4. Enable Accessibility Service when prompted
5. Start dictating anywhere
